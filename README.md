## Setup

```bash
# npm
npm install

# pnpm
pnpm install

# yarn
yarn install
```

## Development

Start the development server on `http://localhost:3000`:

```bash
# npm
npm run dev

# pnpm
pnpm run dev

# yarn
yarn dev
```

## Production

```bash
# npm
npm run build

# pnpm
pnpm run build

# yarn
yarn build
```
